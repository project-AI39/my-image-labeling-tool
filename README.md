# Image Score / Labeling Tool

これはローカルブラウザ上で動作する、画像に対するスコア付け・ラベル付けツールです。

**概要**

- ブラウザ（主に Chromium 系）上で動作する単一ファイルのフロントエンドです。
- 画像フォルダを選択して、各画像に対して定義済みのラベルセットで評価します。
- ラベル定義は JSON 形式で読み込み可能、評価結果はローカルに JSON として保存できます。

**主な機能**

- 画像のズーム・パン
- 未ラベルの画像のみ非表示にするフィルタ
- 自動で次の画像へ移動するオプション
- File System Access API を使った入力／出力フォルダの選択と保存

**注意**

- ブラウザの対応状況に依存します。Chromium 系（Chrome/Edge/Brave 等）での動作が最も安定します。

**使い方（クイックスタート）**

1. リポジトリをクローンまたは `index.html` をダウンロードします。
2. ブラウザで開いた画面からラベル定義 JSON をアップロード（またはサンプルを利用）、
   入力フォルダ（画像）と出力フォルダを選択して「Start」を押します。

**トラブルシューティング**

- ファイル選択が動作しない／書き込み権限エラーが出る場合は、Chromium 系のブラウザで試してください。
